### Typography

#### Existing presentations
- **Headlines:** Playfair Display (serif) — weights 400, 600, 700
- **Body:** Montserrat (sans-serif) — weights 300, 400, 600, 700
- **Digital Pivot deck:** Uses a clean sans-serif (likely system font or Inter) with lime-green accent color on key words

#### Patterns
- Headlines are large and bold (2-4.5rem)
- Body text is lightweight (300-400 weight)
- UPPERCASE small letter-spacing for labels ("TRUST & STABILITY")
- Key words highlighted in accent color (gold or lime-green)
- Italic for quotes/slogans

### Voice & Tone
- **Confident but not arrogant** — "We know hospitality" not "We're the best"
- **Direct, short sentences** — "The fire is lit. The team is ready."
- **Emotional anchors** — heritage (1658), nature, family
- **Pattern:** Bold claim + supporting evidence
- **Key phrases:** "We are clearing the table not to survive, but to build"
- **Numbers speak:** Lead with stats, let data make the case
- **Bilingual:** English primary for international, Finnish for domestic

## 2. LEVI TOUR TEMPLATE ANALYSIS

### Source
"Introduction to Levi as a Destination.pdf" by **Levi Tour** (levitour.fi)
Photos captured from Teams meeting (Patrick, Markus Heiskanen + 1 other)

### Template Structure (11 slides)

| # | Slide Type | Layout | Content |
|---|-----------|--------|---------|
| 1 | **Cover** | Full-bleed aurora photo + centered text band | Destination name (large serif) + tagline |
| 2 | **Welcome/Stats** | Left: hero photo. Right: key numbers grid | 5 headline stats in large type + small labels |
| 3 | **Hero Experience** | Left: text (3 paragraphs). Right: hero photo | Lead attraction deep-dive |
| 4 | **Activities** | Left: title + 2-column bullet list. Right: photo mosaic (6 images) | All available activities |
| 5 | **Accommodation A** | Left: 2 stacked photos. Right: category title + description | Hotels |
| 6 | **Accommodation B** | Same left-photo/right-text layout | Private Chalets |
| 7 | **Accommodation C** | Same layout | Northern Lights Igloos |
| 8 | **Closing** | Full-bleed aurora photo + centered text | "All of this and much more — Lapland awaits!" |
| 9 | **Company** | Centered logo + company description + contact | About the DMC + phone + email |

### Visual Style
- **Background:** Warm cream/beige (NOT white — ~#F5F0EB)
- **Typography:** Elegant serif headlines (similar to Playfair Display), lighter body text
- **Photos:** High-quality, some with rounded corners, mosaic grids for activities
- **Layout:** Clean, generous whitespace, consistent left-right splits
- **Color:** Muted and warm — cream background, dark charcoal text, minimal accent colors
- **Tone:** Aspirational, premium, peaceful — NOT corporate or aggressive

### Key Design Patterns to Replicate
1. **Cover = full-bleed stunning photo + elegant serif text** — sets the emotional tone
2. **Stats page early** — big numbers create credibility fast (#1, 111, 60, 25000, 750000)
3. **Left-right split is the workhorse** — photo(s) left, text right OR text left, photo right
4. **Accommodation categories each get their own slide** — same layout, swappable content
5. **Activity slide uses photo mosaic** — 4-6 photos in a grid, not a single image
6. **Closing = emotional full-bleed** — short aspirational text, no data
7. **Company page last** — logo, description, contact details, call to action

## 3. RECOMMENDED PRESENTATION SYSTEM FOR FINLAND DMC

### Hybrid Approach: Levi Tour elegance + Finland DMC brand

| Element | Levi Tour (adopt) | Finland DMC (keep) |
|---------|-------------------|-------------------|
| Background | Warm cream for content slides | Full-bleed photos for cover/closing |
| Headlines | Elegant serif (Playfair Display) | Already using this |
| Body | Light sans-serif (Montserrat 300) | Already using this |
| Stats | Large numbers with small labels | Add to sales presentations |
| Layout | Left-right photo/text split | Consistent with existing |
| Colors | Warm cream + charcoal | Add FIN+LAND blue as accent |
| Logo | Last slide company page | Bottom-right on every slide |
| Closing | Full-bleed aspirational | Match existing pattern |

### Slide Template Kit (for all presentations)

**T1 — Cover:** Full-bleed Finnish nature photo + FIN+LAND logo centered + presentation title in Playfair Display serif + subtitle

**T2 — Stats/Welcome:** Left hero photo + right side: 4-5 key stats in large Playfair numbers with Montserrat labels

**T3 — Content (text-left):** Left: headline + 2-3 paragraphs. Right: hero photo (rounded corners)

**T4 — Content (photo-left):** Left: 2 stacked photos. Right: headline + text

**T5 — Activity Grid:** Left: headline + bullet list (2 columns). Right: 4-6 photo mosaic

**T6 — Impact Statement:** Full-bleed photo + centered large text (one powerful sentence)

**T7 — Company/Contact:** FIN+LAND logo + company description + team + contact info + "We know hospitality."

### Color Palette for New Presentations
- **Background:** Warm cream #F5F0EB (content slides) / Full-bleed photo (cover/closing)
- **Headlines:** Dark charcoal #1A2634 (Stone Dark)
- **Body text:** Medium charcoal #3D5266 (Stone Light)
- **Accent:** FIN+LAND Blue ~#0035A0
- **Secondary accent:** Gold #C9A227 (for premium feel)
- **Stats numbers:** FIN+LAND Blue or Gold
